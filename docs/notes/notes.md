# Anteckningar, konceptbeskrivning 18/1 

## Ni verkar ha ett särskilt studiekoncept: 

Täning/hälsa-kultur på skolan, uppmanan till att bruka skolans gym och möjlighet att välja deltagande i träningspass eller välja mer idrott på schemat. Idrott kopplad till vanlig undervisning kan förekomma. 


## Konceptbeskrivning - Hemsidan

Beskrivningen stämmer, men är öppen för fri tolkning. *Bra eller dåligt?*

En tydlig "så här tänker vi" som komplettering, där våra lärare berättar om sin syn på texten kanske kan passa? 

Ex: Stresshantering kan betyda olika saker. Man kan jobba aktivit, förebyggande eller som lärare ha det i tankarna under sina lektioner.
